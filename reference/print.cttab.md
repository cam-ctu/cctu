# Print

Print method for \`cttab\`. Long-format objects are first reshaped via
[`cttab_format`](https://cam-ctu.github.io/cctu/reference/cttab_format.md),
then drawn with Unicode box characters. Row indentation is taken from
leading whitespace on the row name as produced by
[`cttab_format`](https://cam-ctu.github.io/cctu/reference/cttab_format.md).

## Usage

``` r
# S3 method for class 'cttab'
print(x, ...)
```

## Arguments

- x:

  Table generated by `cttab`.

- ...:

  Not used.
