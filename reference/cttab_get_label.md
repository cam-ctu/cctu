# Return the variable label of \`x\` if it has one, otherwise \`fallback\`.

Return the variable label of \`x\` if it has one, otherwise
\`fallback\`.

## Usage

``` r
cttab_get_label(x, fallback)
```

## Arguments

- x:

  A vector (typically a column of \`data\`).

- fallback:

  Value to return when \`x\` carries no variable label.
